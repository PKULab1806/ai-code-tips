# AI Code Tips

这个仓库用于在实验室内部收集和整理 `Codex`、`Claude Code` 等 agent 工具的订阅教程、使用经验与相关资料，方便大家快速上手、少走弯路。

我们希望把零散的信息逐步沉淀为一份内部可复用的知识库，包括账号准备、订阅方式、支付流程、常见问题、使用技巧，以及不同工具之间的对比和实践建议。

欢迎大家踊跃贡献内容，一起补充订阅流程、避坑经验、使用心得和实践案例，方便实验室成员交流与参考。

## 参考模板

为了方便大家统一整理内容，仓库提供了一份可直接参考和修改的模板：

- [内容模板](./TEMPLATE.md)

建议新增教程、经验总结或问题排查记录时，先参考这份模板，再按实际情况删减或补充。

## 收集内容

本仓库重点欢迎以下几类内容：

- `Codex`、`Claude Code`、以及其他常见 agent 工具的订阅教程
- 不同平台或地区下的开通流程、支付方式与注意事项
- 订阅过程中常见报错、失败场景与解决办法
- 使用过程中的经验总结、效率技巧和推荐配置
- 多种 agent 工具在功能、价格、稳定性方面的对比
- 适合实验室内部场景的最佳实践与案例分享

## 内容建议

为了便于后续整理，建议在撰写教程或经验时尽量包含以下信息：

- 工具名称与对应版本或套餐类型
- 适用时间与适用地区
- 前置条件，例如账号、支付方式、网络环境等
- 具体操作步骤
- 费用说明与可能的风险点
- 常见问题与排查方法
- 个人体验、优缺点和补充说明

如果某些信息具有时效性，也建议标注记录时间，方便后续更新。

## 贡献方式

欢迎实验室同学直接补充或修改仓库内容。贡献时建议遵循以下原则：

- 内容尽量真实、可复现、可验证
- 教程描述尽量简洁清楚，避免遗漏关键步骤
- 对于可能变化较快的信息，尽量注明时间
- 如果内容来自个人实践，建议补充踩坑记录和注意事项
- 如果不同人对同一工具有不同经验，也欢迎并行记录

## 仓库目标

这个仓库不是为了追求形式上的完整，而是希望持续积累对实验室成员真正有帮助的信息。只要内容能够帮助后来者更快完成订阅、更稳定地使用工具、或者更清楚地理解不同 agent 工具之间的差异，就值得被记录下来。

欢迎持续补充，一起把这份内部资料维护好。
