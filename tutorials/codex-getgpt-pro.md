## 操作步骤

### 1. 准备自己的 OpenAI 账号

首先确保自己已经注册并持有一个可正常使用的 OpenAI 账号。本文的方法默认你是给自己的账号开通相关服务，而不是购买共享账号。

### 2. 进入充值网站

打开 `getgpt.pro`，查找与 Codex 或 OpenAI Plus 相关的充值入口。

如果网站页面分类较多，优先确认以下信息：

- 套餐名称是否对应 `Plus`
- 价格是否与当前页面标注一致
- 充值对象是否为自己的 OpenAI 账号

### 3. 选择套餐并提交信息

根据页面提示选择对应套餐。根据当前记录，`Plus` 套餐价格为 `178 元 / 月`。

提交前建议再次确认：

- 填写的账号信息是否正确
- 套餐周期是否为月付
- 是否会自动续费
- 平台是否提供售后或异常处理渠道

根据当前记录，这种方式`不会自动续费`，到期后如果还想继续使用，需要重新处理下一周期的充值。

### 4. 完成支付并等待到账

按照网站流程完成支付，之后等待服务开通或充值到账。

如果不是即时生效，建议保留支付记录、订单号和页面截图，便于后续排查问题。

### 5. 登录 OpenAI 账号确认状态

支付完成后，重新登录自己的 OpenAI 账号，确认对应服务是否已经生效。

如果短时间内没有到账，可以先检查订单状态，再联系充值平台处理。

## 常见问题

### 这种方式适合谁

- 现象：不确定自己是否适合用第三方充值
- 原因：不同人对稳定性、隐私和支付便利性的要求不同
- 解决办法：如果你已经有自己的 OpenAI 账号，只是缺少方便的支付渠道，这种方式可以作为一个省事的选择

### 充值后没有立刻生效怎么办

- 现象：支付完成后账号状态没有马上变化
- 原因：第三方平台处理存在延迟，或信息填写有误
- 解决办法：先检查订单状态，保留付款凭证，并联系平台客服或售后
